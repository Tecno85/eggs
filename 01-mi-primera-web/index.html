Acerca de mi

!Bienvenido a mí sitio Web ¡

Mi nombre es Iván Madrid, Soy Colombiano, estoy ubicado en la ciudad de Valledupar. Mi meta es ser Desarrollador de Software, tener un conocimiento sólido en Backend y especializarme en el Frontend. En la actualidad estudio Python de manera autodidacta y empecé un Bootcamp con el SENA sobre Programación Básica. Además, estoy en este curso sobre Desarrollo Web en Egg junto a ustedes.

Esta imagen fue creada por Copilot y hace referencia a lenguajes de programación como Python y JavaScript, lenguajes de Hipertexto y de estilos como HTML y CSS. Estos lenguajes son en la actualidad los más demandados por grandes empresas y startups. Si queremos ser desarrolladores con bases sólidas debemos apropiarnos de tecnologías adicionales como Markdown, Git y GitHub.

GitHub es una plataforma en línea para alojar y gestionar proyectos de software, que permite a los desarrolladores colaborar y controlar versiones de su código usando Git.

Markdown es un lenguaje de marcado simple que se usa para formatear texto en documentos y sitios web. Permite agregar elementos como encabezados, listas, enlaces e imágenes con una sintaxis fácil de leer y escribir.

Git es un sistema de control de versiones que permite a los desarrolladores rastrear cambios en su código, colaborar en proyectos y gestionar diferentes versiones del software de manera eficiente.
